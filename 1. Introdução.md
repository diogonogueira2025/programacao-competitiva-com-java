# Introdução
---
### Competitive Programmer's Handbook — Adaptação em Java

Este repositório é uma adaptação do livro _Competitive Programmer's Handbook_ de Antti Laaksonen para o português e para a linguagem de programação Java.

O livro original utiliza C++ como linguagem base. O objetivo aqui não é substituí-lo, mas oferecer uma referência acessível para quem já programa em Java e quer aplicar os conceitos diretamente na linguagem, sem precisar traduzir mentalmente cada exemplo.

### Pré-requisitos

É esperado que o leitor tenha conhecimento prévio de:

- Lógica de programação e algoritmos básicos
- Orientação a objetos e sintaxe Java

### Como usar

Os capítulos seguem a mesma estrutura do livro original. Cada seção apresenta a explicação do conceito seguida de exemplos implementados em Java.

> **Referência original:** LAAKSONEN, Antti. _Competitive Programmer's Handbook_. 2018. Disponível em: [cses.fi/book](https://cses.fi/book/book.pdf)
