# Programação Competitiva em Java 🇧🇷 — Competitive Programmer's Handbook Adaptado

Repositório de **programação competitiva em Java**, baseado no livro _Competitive Programmer's Handbook_.  
Inclui **algoritmos, estruturas de dados, exemplos em Java e explicações em português** para plataformas como Beecrowd, CSES e LeetCode.

---

## 📘 Sobre o projeto

Este repositório é uma adaptação do livro _Competitive Programmer's Handbook_, de Antti Laaksonen, originalmente escrito em C++.

O objetivo é fornecer um material acessível para quem deseja aprender **programação competitiva usando Java**, sem precisar converter mentalmente exemplos em C++.

Aqui você encontrará uma base sólida de **algoritmos, estruturas de dados e técnicas comuns em competições**, com foco em aplicação prática.

---

## 🚀 O que você vai encontrar

- Explicações em português sobre **algoritmos e estruturas de dados**
    
- Exemplos de código em Java para programação competitiva
    
- Padrões de entrada e saída para juízes online (Beecrowd, CSES, LeetCode)
    
- Templates úteis para acelerar a resolução de problemas
    

---

## 🧠 Pré-requisitos

- Lógica de programação e algoritmos básicos
    
- Orientação a objetos e sintaxe Java
    
- Estruturas de dados fundamentais (listas, filas, pilhas, mapas)
