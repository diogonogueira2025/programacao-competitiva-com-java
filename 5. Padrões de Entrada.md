Os padrões de entrada mais comuns em competições são:

---

**1. Valor único**

```java
int n = Integer.parseInt(in.readLine().trim());
```

---

**2. Valores na mesma linha**

```java
// entrada: "3 5 7"
String[] tokens = in.readLine().split(" ");
int a = Integer.parseInt(tokens[0]);
int b = Integer.parseInt(tokens[1]);
int c = Integer.parseInt(tokens[2]);
```

---

**3. N seguido de N linhas (o mais comum)**

```java
// entrada:
// 3
// 10
// 20
// 30
int n = Integer.parseInt(in.readLine().trim());
for (int i = 0; i < n; i++) {
    int x = Integer.parseInt(in.readLine().trim());
}
```

---

**4. N seguido de N valores na mesma linha**

```java
// entrada:
// 3
// 10 20 30
int n = Integer.parseInt(in.readLine().trim());
String[] tokens = in.readLine().split(" ");
for (int i = 0; i < n; i++) {
    int x = Integer.parseInt(tokens[i]);
}
```

---

**5. T casos de teste**

```java
// entrada:
// 2
// 3 5
// 7 8
int t = Integer.parseInt(in.readLine().trim());
for (int i = 0; i < t; i++) {
    String[] tokens = in.readLine().trim().split(" ");
    int a = Integer.parseInt(tokens[0]);
    int b = Integer.parseInt(tokens[1]);
}
```

---

**6. Leitura até EOF**

```java
String linha;
while ((linha = in.readLine()) != null) {
    // processa cada linha até o fim da entrada
}
```

---

**7. Matriz (grid)**

```java
// entrada:
// 3 4
// 1 2 3 4
// 5 6 7 8
// 9 0 1 2
String[] dims = in.readLine().split(" ");
int n = Integer.parseInt(dims[0]);
int m = Integer.parseInt(dims[1]);

int[][] grid = new int[n][m];
for (int i = 0; i < n; i++) {
    String[] row = in.readLine().split(" ");
    for (int j = 0; j < m; j++) {
        grid[i][j] = Integer.parseInt(row[j]);
    }
}
```
